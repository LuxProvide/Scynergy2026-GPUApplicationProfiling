# SCynergy 2026 – GPU Application Profiling with MONAI on Meluxina

This repository serves as the **base code and hands-on material** for the **SCynergy 2026 workshop on GPU application profiling and optimization**, where we use **Meluxina (EuroHPC)** to profile and optimize a **distributed MONAI-based model training**.

The repository is designed to accompany the workshop training material available at:

**https://luxprovide.github.io/Scynergy2026-GPUApplicationProfiling/**

---

## Workshop Context

- **Event:** SCynergy 2026
- **Focus:** GPU application profiling and performance optimization
- **Platform:** **Meluxina** (Luxembourg EuroHPC supercomputer)
- **Application domain:** Medical AI
- **Frameworks & tools:**
  - OpenOnDemand (OOD)
  - **MONAI** (distributed deep learning)
  - **PyTorch (DDP)**
  - NVIDIA NSight Systems

The goal of the workshop is to guide participants through **identifying performance bottlenecks** with provided traces first. Then let the participants generate and analyze traces and apply optimization.

---

## Learning Objectives

By working with this repository, participants will learn how to:

- Profile GPU-accelerated applications
- Analyze:
  - GPU utilization
  - Kernel execution
  - Memory transfers
  - Communication overhead
- Identify performance bottlenecks
- Apply practical optimization techniques for Data loading
